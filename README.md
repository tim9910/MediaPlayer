# 多媒體播放器

## :pushpin: 概述
這是一個簡單的多媒體播放器，使用 C# 和 Windows Forms 開發。提供基本的播放、暫停、停止和選擇檔案功能。用戶可以透過這個播放器來播放視頻檔案，並控制播放過程。

## :pushpin: 開發環境

- **語言**：C#
- **IDE**：Visual Studio 2022
- **平台**：Windows

## :pushpin: 系統操作步驟

- 1. **選擇檔案**  
   點擊下方的「瀏覽」按鈕，選擇你想播放的影片檔案。

- 2. **播放檔案**  
   點擊「播放」按鈕開始播放所選擇的檔案。

- 3. **暫停播放**  
   點擊「暫停」按鈕可以暫停播放，繼續按下「播放」將會從暫停處繼續。

- 4. **停止播放**  
   點擊「停止」按鈕將停止目前的播放。
